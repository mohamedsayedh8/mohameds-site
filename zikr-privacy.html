🔒 Politique de Confidentialité

Dernière mise à jour : 3 mars 2026

Bienvenue sur Zikr. Nous prenons la protection de vos données personnelles très au sérieux. Cette politique explique comment nous collectons, utilisons et protégeons vos informations.

1. Informations collectées

Zikr collecte les informations suivantes :

  • Localisation : utilisée uniquement pour calculer les horaires de prière et la direction de la Qibla. Jamais partagée.
  • Données d'utilisation anonymes : statistiques via Firebase Analytics, sans identification personnelle.
  • Préférences : thème, langue, paramètres — stockés localement sur l'appareil.

2. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Fournir des horaires de prière précis selon votre localisation
  • Personnaliser votre expérience dans l'application
  • Améliorer les fonctionnalités de l'application
  • Envoyer des notifications de rappel (si activé)

3. Données non collectées

Zikr ne collecte jamais :

  • Votre nom ou adresse e-mail
  • Vos contacts ou données d'autres applications
  • Vos données bancaires (gérées par Apple)

4. Services tiers

  • Firebase (Google) : statistiques et synchronisation. Politique Google
  • OpenAI : pour l'IA Imam. Les questions sont transmises à OpenAI. Politique OpenAI
  • Apple StoreKit : gestion des abonnements via l'App Store.

5. Vos droits (RGPD)

Vous avez le droit d'accéder, modifier ou supprimer vos données. Pour toute demande :
