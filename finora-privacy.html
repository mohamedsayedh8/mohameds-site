🔒 Politique de Confidentialité — Finora

Dernière mise à jour : 3 avril 2026

Bienvenue sur Finora. Nous prenons la protection de vos données personnelles très au sérieux. Cette politique explique comment nous collectons, utilisons et protégeons vos informations.

1. Données collectées

  • Compte (optionnel) : email et nom complet si vous créez un compte. L'app fonctionne entièrement sans compte.
  • Transactions financières : montants, catégories, descriptions, dates — saisis manuellement ou via scan de reçu. Ces données vous appartiennent entièrement.
  • Photos de reçus : analysées momentanément par l'IA pour extraire les données, puis supprimées. Aucune image n'est stockée sur nos serveurs.
  • Données de budget et objectifs : configurées par vos soins, stockées localement (mode invité) ou dans notre base sécurisée (compte).
  • Préférences : langue, devise, thème — stockées localement sur votre appareil.

2. Mode invité — Données 100 % locales

Si vous utilisez Finora sans compte, toutes vos données restent exclusivement sur votre appareil. Aucune information n'est transmise à nos serveurs. La suppression de l'app entraîne la perte définitive des données locales.

3. Compte utilisateur — Stockage sécurisé

Si vous créez un compte, vos données sont stockées dans notre base de données sécurisée (Supabase / PostgreSQL) avec chiffrement au repos et en transit (TLS 1.3). Vos données financières ne sont jamais partagées, vendues ou utilisées à des fins publicitaires.

4. Scanner de reçus IA

Les photos de reçus sont envoyées à notre service IA (API sécurisée) uniquement pour extraire les données (montant, marchand, date). Le traitement est instantané et aucune image n'est conservée après analyse. Le service ne peut pas identifier les personnes présentes sur les photos.

5. Abonnement Premium

Les paiements sont gérés exclusivement via Apple App Store ou Google Play. Nous ne collectons ni ne stockons vos informations bancaires ou de carte de crédit. Les reçus d'abonnement sont fournis par Apple/Google.

6. Partage des données

Nous ne vendons, ne louons et ne partageons jamais vos données personnelles avec des tiers à des fins commerciales. Les seuls accès tiers sont :

  • Supabase : hébergement base de données (serveurs UE)
  • API IA : analyse de reçus uniquement (sans stockage)
  • Apple / Google : gestion des achats in-app

7. Sécurité

  • Chiffrement TLS 1.3 pour toutes les communications
  • Authentification sécurisée via Supabase Auth
  • Aucun accès employé aux données financières personnelles
  • Sauvegardes chiffrées régulières

8. Vos droits (RGPD)

Vous disposez des droits suivants :

  • Accès : consulter toutes vos données via l'export dans l'app
  • Rectification : modifier vos données directement dans l'app
  • Suppression : supprimer votre compte et toutes vos données depuis Profil → Supprimer le compte
  • Portabilité : exporter vos données en CSV (Premium)

9. Conservation des données

Vos données sont conservées tant que votre compte est actif. Après suppression du compte, toutes les données sont effacées sous 30 jours. Les données locales (mode invité) sont supprimées immédiatement à la désinstallation.

10. Mineurs

Finora n'est pas destiné aux personnes de moins de 13 ans. Nous ne collectons pas sciemment de données sur les mineurs.

11. Contact

Pour toute question relative à vos données : contact@mohameds.net
